Prediction of Credit Card Fraud, using the data from https://www.kaggle.com/dalpozz/creditcardfraud.

- "Optimization of Logistic Regression" contains a discussion of the unbalanced data, and a grid search for parameter optimization of a Logistic Regression, using ROC-curves. 

- "XGBoost" is a continuation, and performs a grid search for optimal parameters of the XGBoost-algorithm. Two different approaches are taken here: Grid search maximizing the F1-score, and optimization using smoothened ROC-curves.
